Shell completion

Shell completion#

dso supports shell-completion through click.

To enable shell-completion, add the corresponding line to your .bashrc/.zshrc:

# bash
eval "$(_DSO_COMPLETE=bash_source dso)"

# zsh
eval "$(_DSO_COMPLETE=zsh_source dso)"
